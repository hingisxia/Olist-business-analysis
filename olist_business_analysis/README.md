# Olist Business Analysis Project

## Project Overview

This project analyzes the Brazilian Olist e-commerce dataset.

The objective is to build a complete business analysis workflow including:

- Data quality validation
- Data cleaning
- SQLite data warehouse
- Feature engineering
- Business dashboard analysis
- User segmentation (RFM)

## Tech Stack

- Python
- Pandas
- SQLite
- VS Code

## Workflow

Raw Data

↓

↓
